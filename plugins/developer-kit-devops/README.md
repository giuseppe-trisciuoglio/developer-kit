# Developer Kit DevOps

DevOps and containerization expertise.

## Overview

The `developer-kit-devops` plugin provides expertise in DevOps practices, Docker containerization, and GitHub Actions CI/CD pipelines.

## Agents

- **github-actions-pipeline-expert** - GitHub Actions workflows and CI/CD
- **general-docker-expert** - Docker containerization and optimization

## Dependencies

- `developer-kit` (required)
