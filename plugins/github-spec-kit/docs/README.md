## Quick Start

1. **Explore available commands**: See [Command Guide](./guide-commands.md)
2. **Check integration**: Use `/speckit.check-integration`
3. **Optimize workflows**: Use `/speckit.optimize`
4. **Verify specifications**: Use `/speckit.verify`
