## Theming with CSS Variables

Customize by overriding CSS variables in `globals.css`:

```css
@layer base {
  :root {
    --primary: 222.2 47.4% 11.2%;
    --primary-foreground: 210 40% 98%;
    /* ... other variables */
  }
}
```

## Installation Methods

### CLI (Recommended)

```bash
npx shadcn@latest init    # Initialize
npx shadcn@latest add button  # Add component
npx shadcn@latest add --all   # Add all components
```

### Manual

```bash
npm install @radix-ui/react-slot  # Install component dependencies
# Copy component code from ui.shadcn.com into src/components/ui/
```
