# Developer Kit AI

AI/ML capabilities including prompt engineering, RAG, and chunking strategies.

## Overview

The `developer-kit-ai` plugin provides specialized agents and skills for working with AI and machine learning technologies, particularly focused on prompt engineering and Retrieval-Augmented Generation (RAG) systems.


## Agents

- **prompt-engineering-expert** - Prompt design and optimization expertise

## Commands

- `devkit.prompt-optimize.md` - Optimize prompts for specific models and tasks, with detailed output requirements and implementation guidelines

## Skills

- **prompt-engineering** - Prompt design patterns and optimization techniques
- **chunking-strategy** - Document chunking strategies for RAG systems
- **rag** - Retrieval-Augmented Generation implementation patterns


## Dependencies

- `developer-kit` (required)

## Usage

This plugin is commonly used together with:
- `developer-kit-java` - For LangChain4J integration
- Language-specific plugins when building AI-powered applications
