# Developer Kit Python

Python development capabilities.

## Overview

The `developer-kit-python` plugin provides Python development expertise, including code review, refactoring, security, and architecture.

## Agents

- **python-code-review-expert** - Python code review
- **python-refactor-expert** - Python refactoring
- **python-security-expert** - Python security best practices
- **python-software-architect-expert** - Python architecture review

## Skills

- **clean-architecture** - Clean Architecture, Hexagonal Architecture, and DDD patterns for Python
- **aws-lambda-python-integration** - AWS Lambda integration patterns for Python with AWS Chalice Framework and Raw Python handlers

## Dependencies

- `developer-kit` (required)
