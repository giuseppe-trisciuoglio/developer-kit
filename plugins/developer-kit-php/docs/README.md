## About PHP Plugin

The Developer Kit PHP Plugin provides:

- **PHP Agents**: 5 specialized agents for PHP code review, refactoring, security, architecture, and WordPress development
- **PHP Skills**: 1 skill for WordPress Sage theme development

---

## Plugin Structure

```
developer-kit-php/
├── agents/              # PHP development agents
├── skills/              # PHP skills
│   └── wordpress-sage-theme/  # WordPress Sage theme skill
└── docs/               # This documentation
```

---

## Quick Start

1. **Explore available agents**: See [Agent Guide](./guide-agents.md)
2. **Review PHP code**: Use `php-code-review-expert` agent
3. **Develop WordPress themes**: Use `wordpress-development-expert` agent
